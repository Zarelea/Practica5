Bienvenidos a las Tecnologias de Web Frontend..

Las tecnologías de frontend web se refieren a las herramientas y tecnologías utilizadas para crear la interfaz visual y la experiencia del usuario en aplicaciones web



¿Qué son?

Las tecnologías front-end son un conjunto de tecnologías que se utilizan para desarrollar la interfaz de usuario de las páginas web y aplicaciones. Con las tecnologías front-end, los desarrolladores crean todo, desde el diseño y la estructura hasta la animación que vemos en la pantalla al abrir un sitio web, una aplicación web o una aplicación móvil.



¿Qué aprederemos?

  1. Crear interfaces de usuario interactivas y responsivas.
  2. Utilizar HTML, CSS y JavaScript para estructurar y estilizar páginas web.
  3. Optimizar el rendimiento y la accesibilidad de sitios web.
  4. Integrar API y gestionar datos dinámicos en el frontend.
  5. Desplegar y mantener aplicaciones web en diferentes plataformas.


¿Cuales son los navegadores?

Las tecnologías más importantes y populares suelen dividirse en varias categorías se diran las 2 categorias que nos serviran como web frontend, es decir, las tecnologías clave que se utilizan para crear la interfaz de usuario en las aplicaciones web.



    1.Lenguajes de marcados y estilos:

    • HTML (HyperText Markup Language): Es la base del contenido web , proporcionando la estructura de la página web.
    • CSS (Cascading Style Sheets): Se utiliza para el diseño visual y la presentación de una página web.
    • Preprocesadores de CSS: Lenguajes como SASS (Syntactically Awesome Style Sheets) y LESS permiten escribir CSS más eficiente y manejable.


    2.Lenguajes de programación:

    • JavaScript: El lenguaje de programación principal para la interactividad en el navegador.
    • TypeScript: Un superconjunto de JavaScript que añade tipado estático y es especialmente popular en proyectos grandes o con Angular.


Las principales tecnologías utilizadas en el desarrollo frontend son:

  • HTML
  • CSS
  • JavaScript
